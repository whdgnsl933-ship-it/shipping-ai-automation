# 🚢 Shipping AI Automation

## 📌 Overview

This repository contains AI automation workflows for container shipping customer support operations.

The goal is to reduce manual workload and improve response speed and accuracy using Claude and AI agents.

---

## 🧠 Key Functions

* Customer email auto-classification (ETA, BL, Container, Claim)
* Automated response generation
* Schedule inquiry handling
* Container tracking support
* Document request handling (BL copy, invoice)

---

## ⚙️ System Architecture

* Claude (AI reasoning & response generation)
* Automation tools (Make / Zapier / n8n)
* Data sources (Google Sheets / internal systems)

---

## 📂 Structure

* /prompts → AI prompt templates
* /workflows → automation process design
* /scripts → automation scripts (Python, etc.)
* /docs → SOP and manuals

---

## 🎯 Goal

Build a practical AI agent system for real-world shipping operations.
